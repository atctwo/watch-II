◆ getImageData()

imageData watch2::getImageData ( const char *  filename)

get the data of an image as an array of pixels.

this function relies on stb_image.h to return an array of pixel data based on an input file. the image formats supported are dependant on what stb_image.h can handle, which are listed below:

  • JPEG baseline & progressive (12 bpc/arithmetic not supported, same as stock IJG lib)
  • PNG 1/2/4/8/16-bit-per-channel
  • TGA (not sure what subset, if a subset)
  • BMP non-1bpp, non-RLE
  • PSD (composited view only, no extra channels, 8/16 bit-per-channel
  • GIF (*comp always reports as 4-channel, not animated)
  • HDR (radiance rgbE format)
  • PIC (Softimage PIC)
  • PNM (PPM and PGM binary only) This function returns an imageData struct, which contains the raw pixel data, and some metadata, like width and height. Images can be drawn to the screen using drawImage(). Once you have drawn the image, you can free the memory used by calling freeImageData() on the data member of the returned struct.
Parameters
filenamethe path to the image file
Returns
imageData a struct containing image data

◆ getWifiProfiles()

cJSON * watch2::getWifiProfiles ( )

returns the wifi profiles JSON object.

when the system connects to an access point, the connection details are saved in SPIFFS as a "profile", so that the watch can automatically reconnect to it later. this function reads the profiles file from SPIFFS, parses it to create a cJSON struct, and returns it. The profile list can be updated using setWifiProfiles(). If a profiles file doesn't exist, a blank one is created.

The profiles object has two keys:

  • profiles
    • this is an array of objects that each represent one profile. each object has 3 keys:
      • ssid: the access point's SSID
      • password: the access point's password, in plain text...
      • encryption: an enumeration that represents the encryption used by the access point. the possible values can be found here
  • access_list
    • this is an array of strings that represent the SSIDs of profiles in the order of most recently connected.
Returns
cJSON* the wifi profiles
